# soichi-mizutani.com

[soichi-mizutani.com](https://soichi-mizutani.com)

## Development
```
$ git clone https://github.com/lowply/soichi-mizutani.com.git
$ cd soichi-mizutani.com
$ make deps
$ make run
```

Then browse to `http://localhost:9000/`

## Running in production
```
$ docker-compose build
$ docker-compose up -d
```
